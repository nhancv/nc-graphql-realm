# GraphQL + Realm

## Tech stack
### Server
- Nodedemon
- Typescript
- ExpressJs

### Frontend
- React
- Typescript


## Dev
### Server
```
cd server
npm start
```

### Frontend
```
cd frontend
npm start
```

## Build
### Server
```
cd server
npm run build
```

### Frontend
```
cd frontend
npm run build
```
